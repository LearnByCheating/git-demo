# Git-Demo

This project accompanies a series of five CheatSheets and videos covering Beginning and Intermediate Git, Beginning and Intermediate GitHub, and Markdown. The project contains an instructions file for each video listing the steps shown in the video to build this project and integrate Git and GitHub.

### There are 3 components to each topic:  
1. A CheatSheet.
2. A corresponding video that explains the items in the CheatSheet, demonstrating them by building this git-demo project integrating Git and GitHub.
3. An instructions file corresponding to each video listing the steps show. Use this to recreate the project after watching the video. 

---

### Objective:  
The objective is for the student to become familiar with each CheatSheet topic to the point that they can use Git, GitHub, and Markdown:
1. From memory for the most commonly used steps, commands, or syntax.
2. By referencing the CheatSheet for the rest. 

---

### Steps to learn each topic (starting with Beginning Git):
1. Open the **Programming Tools CheatSheet** which includes Git, GitHub, and Markdown.
2. View the **Beginning Git** video.
3. Create your own **git-demo** project following the **v1-beginning-git** instructions file.
    - Build and reset your own git-demo project as many times as you need until you are comfortable with the material. 
    - When can reference the CheatSheet topic and understand what all the items are and how to use them, then move on to the next topic.

Repeat this process until you finish all 5 topics.

Alternatively, you can learn beginning Git, beginning GitHub, and Markdown first. Use those for a while, then come back later to learn Intermediate Git, Intermediate GitHub.
